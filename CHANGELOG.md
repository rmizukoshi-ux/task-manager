# Changelog

## [2026-05-02] — 今日のタスク修正・ボタン名変更

### 変更
- **今日のタスク：納期なしタスクを除外** — 納期が設定されていないタスクは「今日のタスク」に表示しないよう修正。今日以前に納期があるもののみ表示。
- **「タグ管理」→「タグ」** — ヘッダー・タスクモーダルのボタン名を短縮。

## [2026-05-01] — UIリデザイン

### 変更
- **全体UIをリデザイン** — ダークヘッダー＋クリーンライトコンテンツのモノクロ配色に刷新。スレートグレー系カラーパレット、影によるレイヤー表現、洗練されたタイポグラフィ、アニメーション改善。

## [2026-05-01] — Done行グレーアウト

### 変更
- **Doneタスクの行をグレーアウト** — isDone ステータスの行全体を淡いグレー背景に変更。テキストも少し透過して視認性を保ちつつ完了済みを区別。

---

## [2026-05-01] — 機能追加・整理（第2弾）

### 追加
- **タグにナンバー管理** — タグ管理でナンバーを設定可能に。ナンバーが若い順にガントチャートのセクションを並び替え。
- **まとめて追加：タグをタブ選択制に** — ① 共通設定のタグ入力を廃止し、クリックで選択するタグタブ方式に変更。
- **まとめて追加：② フェーズ列を追加** — 各行でフェーズを個別に設定可能に。
- **一括変更：タグごとに全選択** — ガントチャートの各タグセクション左上にチェックボックスを追加。クリックでそのタグの全タスクを選択。
- **一括変更：タグ変更・開始日変更を追加** — 一括変更バーにタグ変更と開始日変更を追加。
- **リンク：ラベル → タイトル（任意）** — リンクの補助入力欄の名称を「ラベル」から「タイトル（任意）」に変更。

### 削除
- **優先度機能を完全削除** — タスクモーダル・ガントチャート・サイドバー・まとめて追加・一括変更・レポートなど全箇所から優先度を削除。

## [2026-05-01] — UX改善

### 変更
- **ガントチャートのタスク列を拡幅** — タスク名が見切れないよう列幅を 240px → 360px に拡大。タスク名にホバーツールチップを追加。
- **フェーズカラー選択をスウォッチ方式に変更** — OSのカラーピッカーを廃止し、12色のプリセットカラーをワンクリックで選択できるパレットに変更。既存フェーズの色変更にも同パレットを使用。

## [2026-05-01] — Project-WBS リリース

### 追加
- **フェーズ機能** — フェーズを自由に作成・管理できる新機能を追加。タスクにフェーズを紐づけ、サイドバーでフィルタリング可能。ガントチャートの各行にフェーズバッジを表示。
- **一括編集バー** — ガントチャートの各行にチェックボックスを追加。複数タスクを選択すると画面下に一括編集バーが表示され、ステータス・優先度・フェーズ・期限日を一括変更、および一括削除が可能。
- **まとめて追加：フェーズ・親タスクの一括設定** — 共通設定欄にフェーズと親タスクの選択を追加。全行に一括適用できる。
- **タグごとにガントチャートを分割** — 1つの大きなガントではなく、タグ単位で独立したガントチャートを表示。
- **日付ヘッダーのスティッキー表示** — ガントをスクロールしても月・日付ヘッダーが常に画面上部に固定表示される。

### 変更
- **名称変更** — 「タスク管理」から「Project-WBS」に変更（タイトル・ヘッダーロゴ）。
- **ヘッダーボタンをテキストのみに** — 全ボタンから絵文字・アイコンを削除（Firebase / ステータス / タグ管理 / フェーズ / まとめて追加 / 今日のタスク / バックアップ / パスワード）。
- **まとめて追加：期限日列を拡幅** — 入力しやすいよう期限日カラムを 115px → 150px に拡大。
- **ステータスポップアップの位置補正** — ガント下部のタスクでステータス変更ポップアップが画面外に出る問題を修正。画面端に達する場合は自動で上方向に表示を反転。
- **Firebase同期にフェーズデータを追加** — phases 配列をFirestore保存・復元の対象に含めた。
- **バックアップにフェーズデータを追加** — JSON バックアップファイルに phases を含めるよう更新（version: 4）。

### 削除
- **単発タスク機能を削除** — 「単発タスク」タグによる分類・専用セクション・サイドバー項目をすべて廃止。全タスクをプロジェクトタスクとして統一管理。
